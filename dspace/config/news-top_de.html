Willkommen bei TUHH Open Research!

Ein Prototyp

Die Daten in diesem System werden nicht dauerhaft gespeichert und können jederzeit geändert oder gelöscht werden!


Was ist das hier:
Dieser Prototyp entsteht im Rahmen des Programms Hamburg Open Science. Ein Repository für Open Access Publikationen wird bis Ende 2018 mit Funktionen für Forschungsinformationssysteme und Forschungsdatenrepositories erweitert.
Umsetzung:
Die technische Umsetzung erfolgt durch die Universitätsbibliothek und das Rechenzentrum der TUHH in Zusammenarbeit mit 4science. Die Software-Basis ist DSpace CRIS Version 5.8
Wie können Sie testen?
Um Materialien hochzuladen nutzen Sie entweder einen TUHH Account oder registrieren Sie sich für einen lokalen Account.
